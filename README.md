## VSCode extension 설치
* C/C++ (ms-vscode.cpptools) → 코드 완성, 디버깅 지원
* CMake Tools (ms-vscode.cmake-tools) (선택) → CMake 프로젝트 관리
* CodeLLDB (macOS/Linux에서 LLDB 디버깅 시 필요)

## CMake shortcuts
* F7 : "CMake: Build"
* Ctrl+Shift+F5 : "CMake: Run Withdout Debugging"

## CMake commands
* cmake --build build
* cmake --build build --target run
